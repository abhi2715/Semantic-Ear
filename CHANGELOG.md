# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Semantic Ear
- Continuous voice listening with real-time detection
- Animated waveform visualization
- Auto-save on silence detection (2 seconds)
- Manual stop & save functionality
- Memory card system with CRUD operations
- Tab-based navigation (Recent Memories, Smart Insights, Timeline)
- Semantic search with keyword matching
- Animated counters that count from 0 to target value on scroll
- Smart insights dashboard with analytics
- Custom cursor with trailing effect and glow
- Glassmorphism design with backdrop blur
- Pastel gradient color scheme (pink, lavender, blue)
- Floating animated background blobs
- Scroll-triggered animations
- Fully responsive design
- Delete memory functionality with smooth animations
- Memory categorization and tagging system
- Timeline view with chronological display
- Search bar with instant results

### Features
- Voice activity detection simulation
- Duration timer for listening sessions
- Real-time waveform animation
- Category-based color coding
- Hover effects on all interactive elements
- Smooth transitions between tabs
- Progressive loading animations

### Design
- Custom typography (Plus Jakarta Sans, DM Sans)
- Rounded corners (2xl+)
- Soft shadows and glows
- Translucent cards with blur effects
- Gradient backgrounds
- Pastel color palette
